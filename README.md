# 🌍 旅遊 AI 平台 - Travel AI Platform

現代化旅遊平台官網 + AI 客服系統，建置於 Vercel 無伺服器架構

## 📋 專案概述

一個功能完整的旅遊平台，提供：
- 🎯 旅遊景點與行程展示
- 🤖 AI 客服諮詢服務
- 📱 完全響應式設計
- ✨ 玻璃擬態視覺設計
- 🚀 Vercel Serverless 部署

## 🚀 快速開始

### 本地開發
```bash
npm install          # 安裝依賴
npm run dev         # 啟動開發伺服器
# 訪問 http://localhost:3000
```

### Vercel 部署
- 推送到 GitHub
- 在 Vercel 連接倉庫
- 自動部署

## 📚 API 端點

- `GET /api/health` - 健康檢查
- `POST /api/chat` - AI 聊天
- `POST /api/contact` - 聯絡表單
